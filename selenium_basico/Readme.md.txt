# Automação de Coleta de Vagas de TI

Projeto em Python utilizando Selenium para automatizar a coleta de vagas de TI
em um portal público de empregos, aplicando filtros por localização
e gerando relatórios em Excel.

## Funcionalidades
- Coleta automática de vagas
- Filtro por São Paulo e Guarulhos
- Extração de título, empresa, local e link
- Geração de planilha Excel com data da coleta

## Tecnologias
- Python
- Selenium
- Pandas
- ChromeDriver

## Objetivo
Projeto desenvolvido para fins de aprendizado e portfólio.
